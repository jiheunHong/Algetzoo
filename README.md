# 🍻 알겠주 (Algetzoo) - 실시간 절주 및 음주 안전 에이전트 MCP 서버

알겠주(Algetzoo)는 음주 전-중-후의 모든 과정을 유기적으로 케어하여, 사용자가 술자리에서 통제력을 유지하고 장기적으로 더 건강한 음주 습관을 형성할 수 있도록 돕는 실시간 안전 개입형 MCP(Model Context Protocol) 서버입니다. 카카오 PlayMCP 등록을 위한 정책들을 준수하여 개발되었습니다.

---

## 💡 주요 모듈 및 기능

| 대화 모듈 | 목적 | 구현된 핵심 기능 | 매핑된 MCP 도구 (Tool) |
| :--- | :--- | :--- | :--- |
| **사전 계획** | 술자리를 시작하기 전 목표를 잡아 위험 줄이기 | 목표 주량 설정, 신체 피로도 및 공복 여부 체크, 비상연락처 설정 | `set_drinking_plan`, `check_condition` |
| **실시간 절주** | 술자리 도중 과음을 인지하고 속도 조절하기 | 음주량 기록, 물 섭취 권장, 음주 속도 실시간 모니터링, 취기 타자 테스트 | `log_drink`, `monitor_pace`, `drunk_self_check` |
| **안전 보호** | 위기 상황에서 벗어나고 안전하게 귀가하기 | 자리 이탈용 가짜 전화 수신 시나리오 제공, 소지품 최종 점검, 카카오택시 호출 | `trigger_escape_call`, `start_safe_return` |
| **사후 회고** | 다음 날 건강한 습관으로 피드백 받기 | 지출 금액 입력, 아침 숙취 컨디션 분석, 절주 달성 피드백 요약 | `generate_recap` |

## 🎨 겹치지 않는 상태별 동물 캐릭터 매핑

각 상태에 맞게 고유하고 직관적인 동물 피드백이 연동됩니다.

1. **음주 페이스 동물 (`monitor_pace`)**
   - 🐢 **거북이 페이스**: 느긋하고 안정적인 템포 (Safe)
   - 🐇 **토끼 페이스**: 조금 깡충깡충 빨라진 템포 (Caution)
   - 🐆 **치타 페이스**: 질주하는 위험한 템포 (Danger)

2. **취기 상태 동물 (`drunk_self_check`)**
   - 🦉 **이성적인 부엉이**: 이성적이고 차분한 맨정신 상태
   - 🦜 **말이 많아진 앵무새**: 술이 들어가며 사교성과 말수가 증가
   - 🐼 **뒤뚱거리는 판다**: 균형 감각이 흐트러지는 어지러운 상태
   - 🦛 **무거워진 하마**: 인지 능력이 크게 둔화되어 기억이 흐려지는 상태
   - 🐕 **네 발로 걷는 멍멍이**: 본능만 남은 만취 상태

3. **아침 숙취 동물 (`generate_recap`)**
   - 🦁 **포효하는 아침 사자**: 숙취 없이 활력이 넘치는 최상의 상태
   - 🐨 **꿀잠 자는 코알라**: 피로를 풀기 위해 충분한 수면이 필요한 상태
   - 🐻 **굴 속의 겨울잠 곰**: 머리가 아파 침대 밖으로 나오기 힘든 상태

## 🚀 로컬 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. FastAPI 서버 구동
```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```
서버 실행 후 브라우저로 `http://localhost:8000`에 접속하여 정상 가동 여부를 시각적으로 확인할 수 있습니다.

### 3. API 상태 확인
*   **헬스체크 및 세션 진단**: `GET http://localhost:8000/health`
*   **MCP SSE 연결 URL**: `GET http://localhost:8000/mcp/sse`

```bash

# 2. 컨테이너 구동 (8000포트 개방)
docker run -d -p 8000:8000 --name algetzoo-mcp algetzoo-mcp-server:latest
```

## 🔒 PlayMCP 심사 정책 준수 사항
*   **평균 응답 속도 100ms 이하**: 인메모리 딕셔너리로 데이터를 저장하여 외부 API 연동 없이 신속하게 응답합니다.
*   **안전한 개인정보 취급**: 주민번호, 카드번호 등의 개인식별정보(PII)를 수집하거나 전송하지 않습니다.
*   **응답 크기 24KB 이하**: 텍스트 형태의 콤팩트한 응답 구조를 제공하여 모바일 네트워크 오버헤드를 방지합니다.
*   **MCP 표준 준수**: 영문 소문자 + 언더바로 구성된 올바른 Tool Name과 명확한 영문 Description을 구성했습니다.
